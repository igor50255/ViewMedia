# VPlay — видеоплеер (C# WPF + WebView2)

## Структура проекта

```
VPlay/
├── VPlay.csproj
├── App.xaml
├── App.xaml.cs
├── MainWindow.xaml
├── MainWindow.xaml.cs
└── player.html          ← кладётся рядом с exe при сборке
```

## Зависимости

- .NET 8 (Windows)
- NuGet: `Microsoft.Web.WebView2` (подтягивается автоматически)
- **WebView2 Runtime** должен быть установлен на машине пользователя:
  https://developer.microsoft.com/ru-ru/microsoft-edge/webview2/

## Сборка

```bash
dotnet build
# или
dotnet publish -c Release -r win-x64 --self-contained false
```

## Функции

| Действие                          | Результат                         |
|-----------------------------------|-----------------------------------|
| Перетащить файл на плеер          | Открыть видео                     |
| `VPlay.exe video.mp4`             | Открыть файл из аргументов + загрузить все видео из папки |
| `Ctrl+D`                          | Полноэкранный режим (любая раскладка) |
| `N`                               | Следующее видео в плейлисте       |
| `B`                               | Предыдущее видео в плейлисте      |
| Кнопка плейлиста (правый верх)    | Открыть/закрыть боковую панель с плейлистом |
| Клик по видео в плейлисте         | Переключиться на выбранное видео  |
| Двойной клик по видео             | Браузерный fullscreen (панель скрывается) |
| `Space`                           | Пауза / воспроизведение           |
| `←` / `→`                        | Перемотка ±5 сек                  |
| `↑` / `↓`                        | Громкость                         |
| `M`                               | Mute                              |
| Клик по видео                     | Пауза / воспроизведение           |
| Завершение видео                  | Автоматический переход к следующему |

## Как работает открытие файла из аргументов

1. C# читает `Environment.GetCommandLineArgs()[1]`
2. Папка файла маппируется на виртуальный хост `vplay-media.local`
   через `SetVirtualHostNameToFolderMapping`
3. JS получает URL вида `https://vplay-media.local/filename.mp4`
4. `<video src="...">` воспроизводит файл напрямую

## Почему Ctrl+D независим от раскладки

В C# перехватываем `OnKeyDown` и проверяем `e.Key == Key.D` —
это логическая клавиша, не зависящая от текущей раскладки клавиатуры.
В JS используем `e.code === 'KeyD'` по той же причине.
