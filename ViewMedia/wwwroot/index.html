Приложение для просмотра видео из Ютуба

  • По умолчанию контент располагается в папке "D:\ContentViewMedia". При изменении пути к папке с контентом, старая папка не удаляется!. Также если по новому пути уже есть папка — ничего не удаляется!.
  • Добавить карточку: скопируйте ссылку на видео с Ютуба и кликните "+ Добавить" в окне программы.
  • Клик по левой половине карточки: открывает видео в браузере, клик по правой: запускает видео в плеере.
  • Правой кнопкой мыши на карточках и пунктах выбора папок вызывается Контекстное меню. Там много функций.
  • Ctrl + D — включает полноэкранный режим и выводит из него. Ctrl + Колёси мыши — увеличивает и уменьшает размер элементов в окне программы.
  • Размер карточек с превью

  • Выбор браузера
